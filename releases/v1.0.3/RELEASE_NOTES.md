# ChromeTech PisoTAB v1.0.3

## Summary
This release publishes updated Android APKs (Hub + Client) with fixes for Hub usability and activation visibility.

## Android Assets
- `android/chrometech-hub-1.0.3-release.apk`
- `android/chrometech-client-1.0.3-release.apk`
- `android/checksums.txt`

## Fixed Issues
- Hub now re-applies fullscreen behavior more reliably in HUB mode.
- Timer Rates page now supports scrolling so Save Rates stays reachable on smaller screens.
- Connected Settings now reflects Portal activation alias updates in Hub.

## OrangePi Compatibility
- For this release, keep using the existing OrangePi image from **v1.0.2** for compatibility:
  - `releases/v1.0.2/orangepi/chrometech-edge-v1.0.2.img.xz`

## Validation
- APK SHA-256 hashes are provided in `android/checksums.txt`.
